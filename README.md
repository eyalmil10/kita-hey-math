# חשבון לכיתה ה׳ (אתר סטטי)

זהו שלד ראשוני לאתר אינטראקטיבי ללימוד חשבון לכיתה ה׳.

## איך פותחים

פשוט פותחים את הקובץ `index.html` בדפדפן.

> אם הדפדפן חוסם פתיחה של חלונות קופצים—לחיצה על “פתחי” פותחת טאבים חדשים, אז צריך לאפשר Popups לאתר המקומי.

## מה יש כרגע

- דף בית עם 4 נושאים:
  - מכנה משותף
  - צימצום שברים
  - ממוצע
  - שברים על ציר המספרים
- לכל נושא נפתח עמוד בטאב חדש (עוד רגע נוסיף גם תרגול ממשי לכל נושא).

## מבנה

- `index.html` – דף הבית
- `styles.css` – עיצוב
- `app.js` – לוגיקה לדף הבית
- `topics/*.html` – עמודי הנושאים
